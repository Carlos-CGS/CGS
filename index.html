World War - Game

Neste jogo, o jogador batalha contra seu oponente recebendo cinco cartas por rodada, devendo escolher qual delas vencerá a carta selecionada pelo oponente. Cada carta possui características únicas, como poder de ataque e tipo, e o resultado do duelo é determinado com base nas regras de vitória e derrota de cada carta. O placar é atualizado a cada rodada, e há efeitos visuais e sonoros que intensificam a experiência. Todo o código foi escrito durante BootCamp da DIO

Desafio do Herói

Este projeto é uma aplicação simples que permite que os usuários classifiquem seus heróis com base na quantidade de experiência (XP) que possuem. O projeto utiliza HTML, CSS e JavaScript. Funcionalidades - Entrada do nome e XP do herói. - Classificação do herói com base na XP fornecida. - Exibição do nível do herói na interface. - Limpeza dos campos após a classificação. - Botão Enter aciona o botão.
Site: Desafio Herói - Repositório: Desafio do Herói

Projeto Pokédex - DIO

Este projeto é uma Pokédex interativa desenvolvida como parte de um desafio da Digital Innovation One (DIO).
A aplicação utiliza a PokeAPI para exibir uma lista de Pokémons e fiz algumas alterações como adicionar detalhamento do pokemon, quando ele é clicado, além de sombras e mouse hover.
