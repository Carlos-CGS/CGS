World War - Game
Neste jogo, o jogador batalha contra seu oponente recebendo cinco cartas por rodada, devendo escolher qual delas vencerá a carta selecionada pelo oponente. Cada carta possui características únicas, como poder de ataque e tipo, e o resultado do duelo é determinado com base nas regras de vitória e derrota de cada carta. O placar é atualizado a cada rodada, e há efeitos visuais e sonoros que intensificam a experiência. Todo o código foi escrito durante BootCamp da DIO
Site: Site World War - Repositório: World War Game

Buscas - AvatarTech
O Assistente de Busca é uma aplicação web que faz buscas em um banco de dados java script e retorna um avatar com as características da linguagem , assim como, uma breve descrição. Possui um direcionamento de propaganda a qual é encviada a propaganda com a linguagem mais pesquisada. Projeto possui responsividade para telas menores e foi criado durante a Imersão Alura X Google.
